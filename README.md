
# Expense Tracker

A full-stack Expense Tracker built using HTML, CSS, JavaScript, Node.js, Express.js, and SQLite.

## Features

* Add Expenses
* Edit Expenses
* Delete Expenses
* Category Management
* Date Tracking
* Expense Summary
* Pie Chart Visualization
* SQLite Database Storage

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* SQLite

## Installation

Clone the repository:

```bash
git clone https://github.com/falak679/expense-tracker.git
```

Install dependencies:

```bash
npm install
```

Run the server:

```bash
node server.js
```

Open:

```text
http://localhost:3000
```
